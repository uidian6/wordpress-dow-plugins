<h1>wordpress-dow-plugins </h1>
<blockquote>
<p>使用方法：下载压缩包>在wordpress后台插件页上传压缩包>安装启用即可（）</p>
<p>目前已知支持WordPress 6.8.3，如大于这个版本请自行测试是否支持</p>
</blockquote>
<h2> 一、功能</h2>
<ul>
<li>1、解析识别网盘</li>
<li>2、显示对应网盘图标</li>
<li>3、支持提取码添加或不添加</li>
<li>4、样式多种渐变颜色（可选）</li> 
<li>5、支持提端设备自适应</li>
